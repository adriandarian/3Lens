## Features Demonstrated

- **Wireframe Mode**: View mesh topology
- **Normal Visualization**: Debug vertex/face normals
- **UV Checker**: Inspect texture coordinates
- **Overdraw View**: Identify overdraw issues
- **Depth Buffer**: Visualize depth values
