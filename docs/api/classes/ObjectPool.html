Class ObjectPool<T>

Generic object pool for reusing objects and reducing GC pressure.

Example

const vectorPool = new ObjectPool({
  factory: () => ({ x: 0, y: 0, z: 0 }),
  reset: (v) => { v.x = 0; v.y = 0; v.z = 0; },
  initialSize: 10,
  maxSize: 100,
});

const vec = vectorPool.acquire();
vec.x = 10;
// ... use vec ...
vectorPool.release(vec);
