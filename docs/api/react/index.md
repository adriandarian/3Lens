# React Bridge API (`@3lens/react-bridge`)

The React bridge provides React hooks and components for integrating 3Lens with React and React Three Fiber applications.

## Quick Links

- [Full React Bridge Documentation →](/api/react-bridge/)
- [React Three Fiber Guide →](/guide/react-r3f)

## Main Exports

### Components

- `ThreeLensProvider` - Context provider for 3Lens
- `ThreeLensCanvas` - Drop-in replacement for R3F Canvas

### Hooks

- `useThreeLensProbe` - Access the probe instance
- `useSelectedObject` - Selection state management
- `useDevtoolEntity` - Register logical entities
- `useMetric` - Access performance metrics
- `useFPS`, `useDrawCalls`, `useTriangles` - Metric shortcuts
