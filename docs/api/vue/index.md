# Vue Bridge API (`@3lens/vue-bridge`)

The Vue bridge provides Vue composables and plugin for integrating 3Lens with Vue and TresJS applications.

## Quick Links

- [Full Vue Bridge Documentation →](/api/vue-bridge/)
- [Vue & TresJS Guide →](/guide/vue-tresjs)

## Main Exports

### Plugin

- `ThreeLensPlugin` - Vue plugin for app-wide integration

### Composables

- `useThreeLens` - Access full devtools context
- `useProbe` - Access the probe instance
- `useSelectedObject` - Selection state management
- `useDevtoolEntity` - Register logical entities
- `useMetric` - Access performance metrics
- `useFPS`, `useDrawCalls`, `useTriangles` - Metric shortcuts
