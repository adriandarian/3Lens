# Angular Bridge API (`@3lens/angular-bridge`)

The Angular bridge provides Angular services, directives, and RxJS observables for integrating 3Lens with Angular applications.

## Quick Links

- [Full Angular Bridge Documentation →](/api/angular-bridge/)
- [Angular Guide →](/guide/angular)

## Main Exports

### Service

- `ThreeLensService` - Main service with RxJS observables

### Module

- `ThreeLensModule` - Angular module with `forRoot`/`forChild`
- `provideThreeLens` - Standalone providers function

### Directives

- `ThreeLensEntityDirective` - Directive for entity registration

### Tokens

- `THREELENS_PROBE` - Injection token for probe instance
- `THREELENS_CONFIG` - Injection token for configuration
