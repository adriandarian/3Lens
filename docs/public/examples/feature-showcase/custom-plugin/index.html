üí° About Custom Plugins
Plugins extend 3Lens with custom panels, toolbar actions, context menu items, and inter-plugin messaging. They have access to frame stats, scene snapshots, and selection state.

Open the 3Lens overlay (Ctrl+Shift+D) to see the plugin panel in action!
